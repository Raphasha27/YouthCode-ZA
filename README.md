## Overview

Curriculum, tools, and platform for teaching programming and AI literacy to young South Africans. Features interactive lessons, project-based learning, and mentorship matching.

## Features

- **Interactive Curriculum** — Python, web dev, and AI fundamentals
- **Project-Based Learning** — Build real-world SA-focused projects
- **Mentorship Matching** — Connect learners with industry professionals
- **Offline Mode** — Learn without internet access
- **Progress Tracking** — Skill assessments and certification paths

## Quick Start

```bash
git clone https://github.com/Raphasha27/YouthCode-ZA.git
cd YouthCode-ZA
pip install -r requirements.txt
python run.py
```
